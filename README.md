## Installation

Clone this repository:

```bash
git clone https://github.com/Death-Mask/catch-kart.git
```

Change to the project directory and run:

```bash
cd catch-kart
chmod +x catch-kart.sh
bash catch-kart.sh
```

On first launch the script will install dependencies automatically.
